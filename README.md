# AI Resume & Portfolio Builder

An AI-powered web application that helps users create professional ATS-friendly resumes and personal portfolios with PDF and Word downloads.

## Features
- AI-generated professional resume summary  
- Multiple ATS-friendly resume templates  
- Profile photo upload  
- Resume preview in real time  
- Download resume as:
  - PDF
  - DOCX (Word)
- AI Portfolio Generator  
- Clean, modern UI  
- Mobile responsive  
- GitHub ready project structure  

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- AI Text Generation API
- python-docx
- FPDF
